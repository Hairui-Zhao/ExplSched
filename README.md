# ExplSched：面向探索性深度学习作业的高效集群调度框架
**对应文章第5章。**
针对现有深度学习集群调度器在处理探索性训练作业时存在的资源浪费和效率低下问题，本项目提出了 ExplSched——一个面向混合作业负载的在线调度框架。探索性作业（如超参数搜索、模型结构调整）常因提前终止而导致大量计算资源被浪费，传统基于 FIFO 或公平分享的调度策略难以适应其动态特性。

ExplSched 将训练过程中损失函数下降幅度（Loss Reduction）定义为作业的“利润”，并以此为导向进行资源分配。该框架基于原始-对偶方法设计在线调度算法，优先将资源分配给在单位资源内能获得更高训练进展的作业，从而在作业提前终止普遍发生的情况下，最大化集群整体效益。
